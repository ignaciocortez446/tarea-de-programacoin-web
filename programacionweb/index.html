¿Por qué aplicar Fibonacci al ahorro?

El principal enemigo del ahorro es la falta de motivación y la monotonía. Los planes de ahorro tradicionales con cuotas fijas suelen abandonarse rápido.

Este sistema utiliza la **Sucesión de Fibonacci** para crear un plan dinámico de acelaración financiera. Al ligar cada depósito mensual a la suma de los dos meses anteriores, el esfuerzo crece de forma escalonada, permitiendo acumular sumas importantes en periodos cortos de tiempo y desafiando la capacidad de generación de ingresos del usuario.

El Algoritmo Detrás del Plan

La lógica matemática toma como base que los dos primeros meses se inicia depositando la unidad básica (1 Bs.). A partir del tercer mes, el sistema ejecuta la fórmula:

Ahorro Actual = Ahorro (Mes Anterior) + Ahorro (Dos Meses Atrás)

El programa realiza este cálculo de forma limpia e iterativa mediante un bucle cerrado, acumulando el dinero de manera progresiva mes a mes sin necesidad de almacenar datos en vectores pesados.
